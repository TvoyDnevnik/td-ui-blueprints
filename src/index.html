Hello, visitor! This page is a showcase of staging TvoyDnevnik UI Library components.

As components get added, they will appear here until eventually ending up in an Angular library which will be used instead of Angular Material in TvoyDnevnik Angular application.

Components:

Button

Input

Known problems:

  1. Label positioning is inaccurate because offsetHeight doesn't get updated to fontSize change. This is working as expected in Angular, so should be fixed in production.
  2. Autocomplete doesn't work reliably. This is because of the of ShadowDOM API by Web Components API. As Angular components utilize virtual DOM, this won't be a problem in production either.
